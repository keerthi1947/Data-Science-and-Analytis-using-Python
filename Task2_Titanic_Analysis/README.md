# Titanic Dataset Analysis

## Data Science Internship - Task 2

This project is part of my Data Science Internship.

In this task, I analyzed the Titanic dataset using Python and performed data cleaning, visualization, and survival analysis.

## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Tasks Performed

* Loaded Titanic dataset
* Cleaned missing values
* Analyzed survival patterns
* Created visualizations using graphs

## Questions Answered

* Who survived more: males or females?
* Did passenger class affect survival?
* What was the survival rate by age group?

## Visualizations

* Survival rate by gender
* Survival rate by passenger class
* Histogram of passenger ages

## Conclusion

From the analysis:

* Females had higher survival rates
* First-class passengers survived more
* Passenger class affected survival chances significantly
